# Consul ECS 0.3.0

## Release Highlights

- **Support file-based configuration :** All CLI flags and options can be passed via a JSON file. This feature also enables manual install of Consul on ECS.

## Support Software Versions

- Consul: 1.10.x, 1.11.x
